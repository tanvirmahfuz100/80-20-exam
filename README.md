# 80/20 Exam Platform

A premium educational platform designed to help students master competitive exams (BCS, SAT, HSC, SSC, etc.) using the 80/20 principle.

## 🚀 Unified Modern Architecture
The project has been consolidated into a single-app architecture to streamline development and move toward a scalable database-driven backend.

- **Frontend**: React + Vite + Tailwind CSS
- **Backend (Planned)**: Supabase (PostgreSQL) for handling 50,000+ questions
- **Design**: Premium "Gamer-Tier" Dark UI with smooth micro-animations

## Project Structure

- `/src`: The core application code (Pages, Components, Context)
- `/public`: Static assets (Initially JSON indices, moving to Storage)
- `implementation_plan.md`: The roadmap for scaling to 50k+ questions

## Next Phases

1. **Supabase Integration**: Connect Authentication and Database.
2. **Dynamic Content**: Migration of JSON data to PostgreSQL.
3. **Analytics**: Real-time strength/weakness tracking.
4. **Social**: Global and Friend Leaderboards.

## Getting Started

```bash
npm install
npm run dev
```
